Practium Project: Question and Answering

The main goal of this project is to generate a set of fluent, reasonable and concise Questions and Answers from a given article.

  • Category: Natural Language Processing
  • Team : 8 members
  • Project duration: 8-weeks

In this sprint, We extracted articles from wikipedia using a web-scraper. From these scrapped html files, we extracted only text into a separate text files.

In this sprint, We focused on the article pre-processing, to find the parts-of-speech(POS) tags, Dependency parsing and Coreference between the words using SpaCy library.

After article pre-processing, it's important to find the patterns of the sentences for framing the questions. To get the patterns, we first manually framed questions and answers for the articles we scraped and checked with the dependency parsing to get them.
For example, to generate binary questions, the proper noun has to be replaced by an auxiliary verb.
